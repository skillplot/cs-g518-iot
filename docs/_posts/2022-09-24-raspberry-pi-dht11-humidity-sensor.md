## Dht11 Humidity Sensor

1. Install libgpiod2
    ```bash
    sudo apt install libgpiod2
    ```
2. Python dependencies
  ```bash
  pip install psutil
  pip install adafruit-python-shell
  pip install adafruit-circuitpython-dht
  ```
